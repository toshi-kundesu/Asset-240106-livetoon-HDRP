## これは何？
- HDRP対応のToonシェーダです
- バーチャルライブ用途に特化させる予定です
- バーチャル照明を当てても破綻しないセルルックを目指すものです

## ライセンス
- 個人法人問わず商用可・改変再配布可
- クレジット記載・ご連絡不要（あると喜びます）
- 詳細:https://unlicense.org/

## 使い方
- AssetsにあるlivetoonディレクトリをUnityプロジェクトにインポートする
- シェーダ名: livetoonをマテリアルに適用する
- Assets/livetoon/Editorも忘れずにインポートする
